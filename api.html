class uwnet.datasets.XRTimeSeries(data)¶

A pytorch Dataset class for time series data in xarray format

This function assumes the data has dimensions [‘time’, ‘z’, ‘y’, ‘x’], and that the axes of the data arrays are all stored in that order.

data : xr.Dataset

>>> ds = xr.open_dataset("in.nc")
>>> dataset = XRTimeSeries(ds)
>>> dataset[0]

class uwnet.model.ForcedStepper(mean, scale, time_step, auxiliary=(), prognostic=(), diagnostic=(), forcing=())¶

Single Column Model Stepper

Integrates the single column equations with the trapezoid rule:

x_n+1 = x_n + f(x_n) dt + (g(t_n) + g(t_n+1)) dt / 2
forward(x, n=None, n_prog=None, dt=None)¶

Produce an n-step prediction for single column mode

x : dict
a dict of torch tensors containing all the keys in the prognostic, diagnostic, and forcing attributes
n : int
the number of time steps to produce a prediction for
n_prog : int
the number of burn-in-steps. The prediction only begins after n_prog steps.

class uwnet.model.SaverMixin¶

Mixin for output and initializing models from dictionaries of the state and arguments

args kwargs

To easily get this to work with a new class add these lines at the top of the __init__ method:

self.kwargs = locals()
self.kwargs.pop('self')
self.kwargs.pop('__class__')
